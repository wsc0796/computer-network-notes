# 3. 使用规则

```text
1. 新截图先按层放入对应文件夹。
2. 每层只维护一份“学习通作业对照.md”。
3. 对照文件只写：题号、考点、答案、已有资料位置、是否需要补强。
4. 不要把学习通题全文重复抄很多遍。
5. 每次新增作业后，更新本总览的“已整理完成”。
```

# 4. 当前最高优先级

```text
1. 运输层：端口16位、TCP/UDP区别、停止等待信道利用率低。
2. 数据链路层：CRC、CSMA/CD、MAC地址、交换机动态学习。
3. 网络层：等新增学习通作业后再单独对照。
```
