# 3. 按目标选择路线

## 10小时速通路线

| 时间 | 做什么 | 资料 |
|---:|---|---|
| 1h | 建立五层模型和核心协议骨架 | `网络速通30题` |
| 1h | 背老师作业高频选择/填空 | `平时作业高频考点提炼` |
| 2h | 练网络层大题：子网、CIDR、RIP、分片 | `大题专项` |
| 1h | 练 TCP/UDP、三握四挥、窗口、HTTP/HTTPS | `TCP_HTTP专题强化` |
| 1.5h | 做模拟卷一、二的客观题并订正 | `模拟卷3套` |
| 0.5h | 做薄弱点综合自测和UDP/编码题 | `薄弱点配套练习` |

| 1h | 回看错题对应笔记 | A层资料 |

适合目标：

```text
从零基础到能做大部分期末题。
```

## 稳80-90路线

```text
1. 先完成10小时速通路线。
2. 2025江西农大B卷闭卷做一遍，优先选择、填空、判断、综合应用。
3. 2021-2022江西农大卷用30分钟做一遍，只查基础漏洞。
4. 大题文件夹7张图全部闭卷做一遍。
5. 模拟卷3套至少做完选择、判断、填空和大题。
6. 薄弱点配套练习至少完成：编码识图、奈奎斯特/香农、GBN/SR、UDP解析、IPv6。
7. 新增运输层学习通作业8题能闭卷说出答案和原因。
8. 把错题按“概念错 / 公式错 / 审题错 / 计算错”分类。
9. 每类错题回到对应专题笔记，补一句自己的错因。
```

额外必须会：

```text
时延计算
CRC
CSMA/CD
以太网最小帧长和MAC地址
IPv4分类/特殊地址/私有地址
子网划分/CIDR/路由聚合
RIP更新
IP分片
ARP/ICMP/NAT
TCP可靠传输、流量控制、拥塞控制
HTTP/HTTPS、DNS、DHCP、POP3/SMTP
```

## 冲90+路线

在稳80-90基础上补：

```text
1. 找更多本校真题或老师题库。
2. 每类计算题至少额外刷5题。
3. 补物理层：奈奎斯特、香农、编码、调制。
4. 补可靠传输：停止等待、GBN、SR。
5. 补应用层细节：FTP、SMTP、POP3/IMAP、DNS递归/迭代、Cookie/缓存。
6. 补网络安全：机密性、完整性、认证、数字签名、报文鉴别。
```

新增的`期末加油站讲义`优先用于第3-5项：

```text
P16-P21：物理层
P29-P30：停止等待、GBN、SR
P32-P38：CSMA/CD、CSMA/CA、以太网、MAC、VLAN
P84-P94：应用层协议
```

## 考研/408级路线

当前资料只能当索引，还要加系统题库：

```text
1. 按章节系统看一轮教材/408网课。
2. 刷王道/天勤网络章节题。
3. 刷历年408真题中的计算机网络部分。
4. 把当前资料作为“快速回忆表”和“期末题型库”。
```

# 5. 每天怎么用

## 一次60分钟学习块

```text
10分钟：读一个专题的速背/模板。
25分钟：做对应题，不翻笔记。
15分钟：对照答案订正。
10分钟：把错因写成一句话。
```

## 一次120分钟学习块

```text
20分钟：看专题笔记。
40分钟：做模拟卷/大题图。
30分钟：订正并回笔记。
20分钟：复做错题。
10分钟：闭卷默写模板。
```

## 每天结束前

只检查 4 件事：

```text
1. 今天做了几题？
2. 错题属于哪类？
3. 哪个模板还背不出来？
4. 明天先复做哪3题？
```

# 6. 新资料入库规则

以后新增截图、作业、真题、PDF讲义，按这个流程：

```text
1. 放到对应目录：学习通作业 / 大题 / 往年卷线索 / 06_新增资料。
2. 先做 contact sheet 总览。
3. 判断它属于哪类题型。
4. 如果笔记已有模板，不补笔记，只加入题型索引。
5. 如果笔记没有模板，补到对应专题文件。
6. 最后加入模拟/大题训练顺序。
```

不要做：

```text
看到新资料就全文抄进笔记。
看到新课就完整看完。
同一个知识点在多个文件重复扩写。
```

```text
这是题目/截图：
我做到哪一步：
我卡在哪里：
请判断笔记里有没有，若没有请补。
```

```text
1. 先查笔记。

3. 无模板：补进对应专题文件。
4. 最后给考试记法。
```

# 8. 闭卷默写清单

## 基础概念

```text
OSI七层
TCP/IP四层或五层
每层PDU
协议/服务/接口区别
发送时延、传播时延
```

## 数据链路层

```text
数据链路层三大问题
CRC步骤
PPP不提供可靠传输
CSMA/CD工作过程
争用期=2t
MAC地址48位
以太网最小帧64B，数据字段最小46B
交换机MAC表动态学习
```

## 网络层

```text
IPv4分类和默认掩码
特殊地址：0.0.0.0、127.0.0.1、255.255.255.255
私有地址三段
子网掩码和可用主机数
网络地址、广播地址、可用主机范围
路由聚合最长公共前缀
最长前缀匹配
RIP更新规则
IP分片：MTU、MF、片偏移/8
ARP、ICMP、NAT作用
```

## 运输层

```text
TCP vs UDP
端口号、复用与分用
TCP首部关键字段
三次握手
四次挥手
可靠传输：序号、确认、累计确认、重传、SACK
发送窗口=min(rwnd,cwnd)
流量控制 vs 拥塞控制
慢开始、拥塞避免、快重传、快恢复
超时后 ssthresh=cwnd/2，cwnd=1
```

## 应用层

```text
DNS解析过程
HTTP vs HTTPS
HTTP默认80，HTTPS默认443
DHCP DORA
SMTP发邮件，POP3/IMAP收邮件
```
