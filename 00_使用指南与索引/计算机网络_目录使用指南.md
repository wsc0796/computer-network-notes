# 4. 做题顺序

## 4.1 选择/判断

先刷：

```text
计算机网络_模拟卷3套_含详细解析.md
```

刷法：

```text
第一遍遮住答案。
第二遍只看错题解析。
第三遍把错题对应知识点回到专题笔记里标出来。
```

## 4.2 简答题

优先背这些模板：

```text
1. 协议与服务区别。
2. 数据链路层三个基本问题。
3. TCP三次握手过程及原因。
4. TCP四次挥手过程及原因。
5. 流量控制与拥塞控制区别。
6. HTTP与HTTPS区别。
7. 交换机与路由器区别。
```

## 4.3 计算题

优先练：

```text
1. 发送时延 = 数据长度 / 发送速率。
2. 传播时延 = 距离 / 传播速率。
3. 传输效率 = 有效数据 / 总数据。
4. CRC余数。
5. CSMA/CD争用期 = 2t。
6. 发送窗口 = min(rwnd,cwnd)。
7. 拥塞控制 cwnd 增长。
8. 奈奎斯特/香农。
9. GBN/SR窗口与重传。
10. UDP首部十六进制解析。
```
