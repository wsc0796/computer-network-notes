## 1. 基带信号（作业原题考点）

```
基带信号 = 来自信源的原始信号。
```

两个易错点：
- 基带信号**不等于**数字信号——模拟信源（如麦克风）产生的也是基带信号
- 基带信号含低频/直流成分，**很多信道不能直接传输** → 需要调制

---

## 2. 数据通信系统模型

```text
源点 → 发送器 → 传输系统 → 接收器 → 终点
│        │          │          │        │
└─源系统─┘          │          └─目的系统─┘
              传输系统
```

三个组成部分：**源系统**、**传输系统**、**目的系统**。

---

## 3. 基带调制 vs 带通调制

| | 基带调制（编码） | 带通调制 |
|:--|:--------------|:--------|
| 是否用载波 | ❌ 不用载波 | ✅ 用高频载波 |
| 做什么 | 改变数字信号的**波形** | 将基带信号的频率**搬移**到高频 |
| 输出 | 仍是基带信号 | 带通信号 |
| 例子 | NRZ、Manchester | ASK、FSK、PSK、QAM |
| 适用场景 | 近距离基带传输 | 远距离/无线/频分复用 |

一句话区分：

> 基带调制 = 给信号"整形"；带通调制 = 给信号"搬家"。

---

## 4. 带通调制四种方法

| 方法 | 全称 | 改变载波的哪个参数 |
|:-----|:-----|:-----------------|
| **ASK** | 幅移键控 | **振幅** |
| **FSK** | 频移键控 | **频率** |
| **PSK** | 相移键控 | **相位** |
| **QAM** | 正交振幅调制 | **振幅 + 相位** |

### 考试必背

> FSK改变的是频率。QAM同时改变振幅和相位。

---

## 5. 基带传输 vs 带通传输

| | 基带传输 | 带通传输 |
|:--|:-------|:-------|
| 传输内容 | 基带信号（不经载波调制） | 调制后的带通信号 |
| 代表 | 10Base-T 以太网 | WiFi、4G/5G、卫星通信 |
| 10Base命名 | "Base" = 基带传输 | "Broad" = 宽带传输 |
