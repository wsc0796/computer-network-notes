## 2. 计算题专项：9类必练

OCR 明确识别出计算题专项的 9 类题型：

| 优先级 | 题型 | 你要会什么 |
|---|---|---|
| P1 | 子网划分 | 网络地址、广播地址、可用主机数、定长/变长划分 |
| P1 | IPv4分片 | MTU、首部长度、每片数据长度、片偏移、MF |
| P1 | 路由聚合 | 写二进制、找最长公共前缀、CIDR表示 |
| P1 | TCP拥塞控制 | cwnd、ssthresh、慢开始、拥塞避免、超时/重复ACK |
| P2 | 停止等待/GBN/SR | 信道利用率、窗口大小、序号位数 |
| P2 | CSMA/CD | 最小帧长、争用期、传播时延、最远距离 |
| P2 | 奈奎斯特/香农 | 速率上限、信噪比dB换算、调制状态数 |
| P2 | 电路/报文/分组交换 | 存储转发时间、分组大小、链路速率 |
| P3 | 路由算法与协议 | 距离向量、RIP更新、OSPF/Dijkstra |

### 你现在最该补的计算顺序

```text
1. 子网划分
2. IPv4分片
3. 路由聚合
4. TCP拥塞控制
5. GBN/SR窗口和序号位数
6. CSMA/CD最小帧长/距离
7. 奈奎斯特/香农
```

## 7. 第5章：运输层

OCR 提取出的高频考点：

```text
UDP协议
TCP段
TCP连接管理
TCP可靠传输
TCP流量控制
TCP拥塞控制
```

真题 PPT 最后页判断：

```text
传输层的考点非常集中，只有UDP和TCP两个协议，以TCP为重中之重。
```

期末必背模板：

```text
TCP三次握手：SYN -> SYN+ACK -> ACK。
TCP四次挥手：FIN -> ACK -> FIN -> ACK。
TCP确认号：期望收到的下一个字节序号。
可靠传输：序号、确认、重传、滑动窗口。
流量控制：保护接收方，rwnd。
拥塞控制：保护网络，cwnd。
发送窗口 = min(rwnd, cwnd)。
```

```text

HTTP：无状态，请求/响应，80端口。
HTTPS：HTTP + TLS，443端口，提供加密和身份认证。
FTP：控制连接和数据连接分离。
邮件：SMTP发送，POP3/IMAP接收。
```
