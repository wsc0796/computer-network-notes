# 08 大题题型与解题步骤

## 来源信号

```text
奈奎斯特/香农

```

## 1. 子网划分

```text
1. 看题目给的是前缀还是主机数。
2. h = 32 - 前缀长度。
3. 地址总数 = 2^h，可用主机数 = 2^h - 2。
4. 块大小 = 256 - 掩码最后一个非255字节。
5. 网络地址按块大小递增。
6. 广播地址 = 下一个网络地址 - 1。
```

## 2. IPv4分片

```text
1. 原始数据长度 = IP总长度 - IP首部长度。
2. 每片最大数据长度 = floor((MTU - IP首部长度) / 8) * 8。
3. 除最后一片外，每片数据长度必须是8字节整数倍。
4. 片偏移 = 数据起始位置 / 8。
5. 最后一片 MF=0，其余 MF=1。
```

## 3. 路由聚合

```text
1. 把多个网络地址写成二进制。
2. 找最长公共前缀。
3. 公共前缀后全部置0。
4. 写成 CIDR 形式。
```

## 4. RIP更新

```text
1. 收到邻居通告距离后，先全部 +1。
2. 原来没有该目的网络：加入。
3. 原下一跳就是该邻居：必须更新。
4. 原下一跳不是该邻居，但新距离更短：更新。
5. 距离 >=16：不可达。
```

## 5. TCP拥塞控制

```text
慢开始：cwnd从小开始指数增长。
达到ssthresh后进入拥塞避免：线性增长。
超时：ssthresh通常变为当前窗口一半，cwnd重新变小。
三次重复ACK：快重传、快恢复。
```
