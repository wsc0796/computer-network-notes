## DNS

```text
作用：域名解析为IP地址。
常用端口：UDP 53。
解析链路：缓存 -> 本地DNS -> 根DNS -> 顶级域DNS -> 权威DNS。
```

易错：

```text
根DNS通常不直接返回最终IP。
递归查询和迭代查询要区分。
```

## HTTP/HTTPS

```text
HTTP：明文，默认80端口。
HTTPS：HTTP + TLS，默认443端口。
HTTP是无状态协议，可用Cookie维持状态。
```

## 邮件与FTP

```text
SMTP：发送邮件，常见25端口。
POP3：下载式收邮件，常见110端口。
IMAP：在线管理邮件，常见143端口。
FTP：控制连接21，数据连接与主动/被动模式有关。
```

## DHCP

```text
Discover -> Offer -> Request -> ACK
用于动态分配IP地址、网关、DNS等网络配置。
```
