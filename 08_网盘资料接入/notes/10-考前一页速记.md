# 10 考前一页速记

## 数字

```text
IPv4 32位，IPv6 128位
MAC 48位
端口号 16位
HTTP 80，HTTPS 443，DNS UDP 53
以太网最小帧 64B
RIP 15跳最大，16不可达
```

## 公式

```text
发送时延 = 数据长度 / 发送速率
传播时延 = 距离 / 传播速度
奈奎斯特：C = 2W log2(V)
香农：C = W log2(1 + S/N)
可用主机数 = 2^(32-前缀) - 2
发送窗口 = min(rwnd, cwnd)
```

## 网络层大题

```text
子网：算主机位、块大小、网络地址、广播地址。
CIDR：多条匹配选最长前缀。
分片：每片有IP首部，偏移单位8B，最后MF=0。
RIP：邻居距离+1，16不可达。
```

## TCP

```text
三次握手：SYN -> SYN+ACK -> ACK。
四次挥手：FIN -> ACK -> FIN -> ACK。
TIME_WAIT：保证最后ACK到达，让旧报文消失。
TCP可靠传输：序号、确认、重传、滑动窗口。
流量控制保护接收方，拥塞控制保护网络。
```

## 链路层与应用层

```text
CRC：生成多项式转二进制，补r个0，模2除，余数接后面。
ARP：请求广播，响应单播，跨网查网关MAC。
DNS：缓存 -> 本地DNS -> 根 -> TLD -> 权威。
HTTP明文，HTTPS=HTTP+TLS。
DHCP：Discover -> Offer -> Request -> ACK。
```
