# 06 选择判断题高频考点

## 1. 数字类

```text
MAC地址：48位。
IPv4地址：32位。
IPv6地址：128位。
端口号：16位。
HTTP：80。
HTTPS：443。
DNS：UDP 53。
以太网最小帧：64字节。
RIP最大有效跳数：15，16表示不可达。
```

## 2. 协议归属

```text
交换机：数据链路层。
路由器：网络层。
TCP/UDP：运输层。
HTTP/DNS/FTP/SMTP：应用层。
ARP：网络层与链路层之间，常和局域网一起考。
ICMP：网络层辅助协议。
```

## 3. 单播/广播

```text
ARP请求：广播。
ARP响应：单播。
DHCP Discover：常用广播。
跨网通信：帧目的MAC是下一跳网关MAC，不是最终主机MAC。
```

## 4. 路由

```text
最长前缀匹配：多条匹配时选前缀最长。
RIP：距离向量。
OSPF：链路状态，Dijkstra。
BGP：路径向量，AS之间。
```

## 5. TCP

```text
TCP确认号表示期望收到的下一个字节序号。
TCP面向字节流，UDP面向报文。
主动关闭方进入TIME_WAIT。
TIME_WAIT等待2MSL。
发送窗口 = min(rwnd, cwnd)。
```
