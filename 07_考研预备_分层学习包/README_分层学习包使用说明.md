# 计算机网络考研预备：分层学习包使用说明

本目录用于大二下阶段把计算机网络整理成后续考研可复用资料。

每一层固定保留 4 份笔记：

```text
01_零基础学习版.md      (或 02_公式与性能指标.md 用于公式密集型章节)
02_课堂截图对应笔记.md   (或 03_易错概念对比.md 用于概念密集型章节)
03_原资料包对照补强.md
04_题型训练.md
```

总错题统一放在：

```text
00_总错题表/计算机网络_总错题表.md
```

网络概述与体系结构（408第1章）作为独立前置包：

```text
00_网络概述与体系结构/
├─ 01_零基础学习版.md
├─ 02_公式与性能指标.md
├─ 03_易错概念对比.md
└─ 04_题型训练.md
```

## 使用顺序

每学一层都按这个流程：

```text
0. 先完成 00_网络概述与体系结构（无截图要求，直接阅读+做题）
1. 看视频并截图
2. 生成课堂截图对应笔记
3. 生成零基础学习版
4. 和原资料包对照，生成补强笔记
5. 做题型训练
6. 错题进入总错题表
```

## 层级顺序

建议按自顶向下（前置：网络概述）：

```text
00_网络概述与体系结构  ← 先完成，建立全局认知
01_应用层
02_运输层
03_网络层
04_数据链路层
05_物理层
```

```text

能知道考研常考题型
```
