# 网络概述 —— 公式与性能指标

> 408选择题、计算题高频来源
> 必须默写：四种时延公式 + 时延带宽积 + 利用率关系

---

## 1. 带宽（Bandwidth）

### 两种含义（区分清楚）

| 含义 | 应用场景 | 单位 |
|:-----|:---------|:-----|
| **频域带宽** | 物理层：信号占用的频率范围 | Hz |
| **数据率** | 网络层及以上：数字信道传送数据的速率 | bps |

在计算机网络中，**带宽通常指数据率**，即信道能传输的最大比特率。

---

## 2. 四种时延

### 总时延公式

```text
总时延 = 发送时延 + 传播时延 + 处理时延 + 排队时延
```

### 逐一拆解

| 时延类型 | 含义 | 公式 | 谁决定 |
|:---------|:-----|:-----|:-------|
| **发送时延** | 把数据推到链路上所需时间 | 分组长度 ÷ 发送速率 | 发送速率 |
| **传播时延** | 信号在链路上传播的时间 | 信道长度 ÷ 传播速率 | 距离 |
| **处理时延** | 路由器处理分组的时间 | 题目通常给出 | 路由器性能 |
| **排队时延** | 分组在队列中等待转发的时间 | 题目通常给出 | 当前网络拥塞 |

### 发送时延 vs 传播时延（必考区分）

```
发送时延：水管口子的大小 —— 把一桶水倒进水管要多久
传播时延：水管的长度 —— 水滴从A端流到B端要多久
```

两者**没有因果关系**。传播时延取决于距离和信号传播速度，不取决于数据量。

### 例题1：基本计算

> 收发两端相距1000km，信号在介质中传播速率为 2×10⁸ m/s，数据长度 10⁷ bit，发送速率为 100 kbit/s。求发送时延和传播时延。

```
发送时延 = 10⁷ / (100×10³) = 100 s
传播时延 = 1000×10³ / (2×10⁸) = 10⁶ / (2×10⁸) = 0.005 s = 5 ms
```

**结论**：此场景下发时延远远大于传播时延。总时延 ≈ 发时延 = 100 s。

### 例题2：哪段是瓶颈

> 收发两端距离1000km，信号传播速率 2×10⁸ m/s，数据长度为 4000 bit，发送速率为 1 Mbps。求发送时延和传播时延。

```
发送时延 = 4000 / 10⁶ = 0.004 s = 4 ms
传播时延 = 10⁶ / (2×10⁸) = 0.005 s = 5 ms
```

**结论**：此场景下两者量级相同。

## 3. 时延带宽积

```text
时延带宽积 = 传播时延 × 带宽
```

**物理含义**：在第一个比特到达接收方之前，发送方已经发出去了多少个比特。即**正在链路上"飞行"的比特数**。

### 例题3

> 传播时延为20ms，带宽为100Mbps，求时延带宽积。

```
时延带宽积 = 20×10⁻³ × 100×10⁶ = 2×10⁶ bit = 2 Mbit
```

> 这意味着：发送方发出第一个比特后，在它到达接收方之前，发送方还能继续发出 2 Mbit 的数据。

---

## 4. RTT（往返时间）

```text
RTT = 往返时间 = 数据从发送方到接收方再返回的总时间
```

RTT通常用于TCP超时计算，包含了两段传播时延 + 沿途路由器的处理时延。

### 例题4

> 卫星通信：地球站到卫星的距离为36000 km，信号在自由空间以光速 3×10⁸ m/s 传播。求单程传播时延和RTT。

```
单程传播时延 = 36000×10³ / (3×10⁸) = 120 ms
RTT = 120 × 2 = 240 ms（尚未计算处理时延）
```

---

## 5. 吞吐量（Throughput）

```text
吞吐量 = 单位时间内成功传输的数据量
```

| 概念 | 含义 |
|:-----|:-----|
| **即时吞吐量** | 某一时刻的瞬时速率 |
| **平均吞吐量** | 一段时间内的平均速率 |

实际吞吐量受限于瓶颈链路（速率最小的链路）。

### 例题5

> 路径由三段链路组成：R₁=2Mbps, R₂=3Mbps, R₃=5Mbps。求该路径的最大吞吐量。

```
最大吞吐量 = min(2, 3, 5) = 2 Mbps（瓶颈为R₁）
```

## 6. 利用率

```text
信道利用率 = (数据通过的时间) / (总时间)

在排队论中，当利用率趋近于1时：
排队时延 → ∞
```

### 利用率与时延的关系公式

```text
D = D₀ / (1 - U)

D₀ = 网络空闲时的时延（传播时延+处理时延）
U = 利用率
D = 总时延
```

当U从0.5→0.9时，D从2D₀→10D₀。
当U→1时，D→∞（网络崩溃）。

---

## 7. 传输效率（有效数据率）

```text
传输效率 = 有效数据 / 总发送数据 × 100%
```

### 例题6

> 应用层数据 400B，经过运输层（加20 B首部）→ 网络层（加20 B首部）→ 数据链路层（加20 B首部 + 4 B尾部）。求传输效率。

```
总发送数据 = 400 + 20 + 20 + 20 + 4 = 464 B
传输效率 = 400 / 464 × 100% ≈ 86.2%
```

## 8. 必须记住的公式汇总

| 名称 | 公式 | 单位 |
|:-----|:-----|:-----|
| 发送时延 | 数据长度 ÷ 发送速率 | s |
| 传播时延 | 信道长度 ÷ 传播速率 | s |
| 总时延 | 发送时延 + 传播时延 + 处理时延 + 排队时延 | s |
| 时延带宽积 | 传播时延 × 带宽 | bit |
| 利用率 | T_data / T_total | 无单位 |
| 传输效率 | 有效数据 ÷ 总数据 | 百分比 |
| 吞吐量 | min(各段链路带宽) | bps |
| RTT | 2 × 传播时延 + 处理时延 | s |

## 9. 光速/电信号传播速度速记

| 介质 | 传播速率 | 备注 |
|:-----|:---------|:-----|

## 10. 自测题

1. 带宽为 10 Mbps，发送 1000 Byte 的数据，发送时延 = ？
2. 两地相距 200 km，信号传播速率 2×10⁸ m/s，传播时延 = ？
3. 传播时延 50 ms，带宽 1 Gbps，时延带宽积 = ？
4. 路径由 R₁=10 Mbps、R₂=5 Mbps、R₃=20 Mbps 组成，最大吞吐量 = ？
5. 有效数据 800 B，总发送 1000 B，传输效率 = ？

<details>
<summary>答案</summary>

1. 1000×8 / (10×10⁶) = 8000 / 10⁷ = 0.0008 s = **0.8 ms**
2. 200×10³ / (2×10⁸) = 10³ / 10⁶ = **0.001 s = 1 ms**
3. 50×10⁻³ × 1×10⁹ = **50 Mbit**
4. **5 Mbps**（取最小）
5. 800 / 1000 × 100% = **80%**

</details>
