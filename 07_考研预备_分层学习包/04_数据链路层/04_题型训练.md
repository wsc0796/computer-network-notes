# 数据链路层：题型训练

## 1. 题型总表

| 题型      | 高频考点                  | 掌握标准 |
| ------- | --------------------- | ---- |
| 概念判断    | 三大问题、PPP、MAC、交换机      | 不背反  |

| GBN/SR  | 滑动窗口、重传范围             | 会判断  |

## 2. 必背速记

```text
数据链路层 PDU 是帧。
三个基本问题：封装成帧、透明传输、差错检测。
CRC 用于检错，不负责纠错。
PPP 不保证可靠传输。
MAC 地址通常 48 位。
交换机工作在数据链路层。
交换机通过源 MAC 地址动态学习 MAC 表。
CSMA/CD：先听再发，边发边听，冲突停止，随机退避。
```

## 3. CRC 做题模板

```text
1. 生成多项式转二进制。
2. 原数据后补 r 个 0，r = 生成多项式位数 - 1。
3. 模 2 除法。
4. 余数不足 r 位就在前面补 0。
5. 原数据 + 余数 = 发送序列。
```

## 4. 简答模板

### 交换机工作原理

```text
交换机工作在数据链路层，根据 MAC 地址转发帧。
交换机收到帧后，根据源 MAC 地址和入端口动态学习 MAC 表；
转发时根据目的 MAC 地址查表，若找到则从对应端口转发，若找不到则泛洪。
```
