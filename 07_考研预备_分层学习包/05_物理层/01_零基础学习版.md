## 1. 物理层解决什么问题

物理层解决的是：

```text
如何把比特 0/1 转换成物理信号，并在传输介质上传输。
```

它不关心数据代表什么含义，只关心：

```text
怎么编码
怎么发送
能传多快
能传多远
噪声会造成什么限制
```

## 2. 物理层主线

```text
比特
    ↓
编码成信号
    ↓
通过介质传输
    ↓
受带宽、噪声、信噪比限制
    ↓
用奈奎斯特/香农公式估算极限速率
```

## 3. 比特率与波特率

比特率：

```text
每秒传输多少比特，单位 bps。
```

波特率：

```text
每秒传输多少码元，单位 Baud。
```

关系：

```text
若每个码元携带 n bit，则比特率 = 波特率 * n。
```

## 4. 编码方式

已有资料重点补强四种：

| 编码 | 识别要点 |
|---|---|
| NRZ | 高低电平直接表示 0/1，中间不一定跳变 |
| NRZ-I | 遇到 1 翻转，遇到 0 不翻转 |
| 曼彻斯特 | 每个比特中间都有跳变 |
| 差分曼彻斯特 | 每个比特中间有跳变，位开始是否跳变表示数据 |

考试重点：

```text
看波形识别编码类型。
```

## 5. 10Base 命名

常见记忆：

```text
10 表示 10 Mbps。
Base 表示基带传输。
T 表示双绞线。
5 表示粗缆约 500m。
2 表示细缆约 185m。
```

## 6. 奈奎斯特公式

适用：

```text
无噪声、理想低通信道。
```

公式：

```text
C = 2W * log2(V)
```

其中：

```text
W 是带宽。
V 是码元离散电平数。
```

## 7. 香农公式

适用：

```text
有噪声信道。
```

公式：

```text
C = W * log2(1 + S/N)
```

如果给的是 dB：

```text
SNR(dB) = 10 * log10(S/N)
```

常见：

```text
30 dB -> S/N = 1000
```
