# 运输层与网络层：阶段抽查题

> 用途：6小时计划中的阶段预检。  
> 规则：闭卷完成；做完后再看本文件对应答案。  
> 注意：本抽查题与正式验收卷使用不同数值和题目，不能替代最终60分钟验收。

## A. 运输层抽查（13分钟，5题）

### 1. TCP/UDP 与分用（选择，2分）

关于传输层分用，以下说法正确的是（ ）。

A. UDP 只根据目的 IP 地址分用  
B. TCP 连接通常由源 IP、源端口、目的 IP、目的端口四元组标识  
C. TCP 和 UDP 都不需要端口号  
D. 同一主机上不同进程不能使用不同端口

### 2. 可靠传输分类（填空，2分）

可靠传输协议可分为单帧滑动窗口和多帧滑动窗口。停止-等待 ARQ 属于 ________；GBN 和 SR 属于 ________。

### 3. TCP 序号确认号（计算，3分）

主机 A 发送 TCP 数据段，`seq=2500`，携带 `420` 字节数据。B 正确收到后回复的 `ack` 是多少？若 A 发送不携带数据的 FIN，`seq=6000`，B 回复 `ack` 是多少？

### 4. RTT/RTO（计算，3分）

旧 `SRTT=80ms`，旧 `RTTVAR=16ms`，新样本 `R=120ms`，`alpha=1/8`，`beta=1/4`。按 RFC 6298 顺序计算新 `RTTVAR`、新 `SRTT`、`RTO=SRTT+4*RTTVAR`，忽略 `G` 和最小 RTO。

### 5. 拥塞控制（判断，2分）

判断并改正：TCP 流量控制由 `cwnd` 决定，拥塞控制由 `rwnd` 决定。（ ）

## B. 网络层抽查（10分钟，5题）

### 6. 非大题概念组（判断，2分）

判断并改正：数据报网络通信前必须建立虚电路；虚电路网络每个分组都独立查路由表。（ ）

### 7. 转发、路由、直接交付/间接交付（填空，2分）

转发是路由器根据 ________ 对单个分组选择输出端口；路由是通过路由协议生成和维护 ________。若目的主机与源主机在同一网络，属于 ________ 交付；否则先交给默认网关，属于 ________ 交付。

### 8. IP 与 MAC（简答，2分）

主机 `192.168.1.10/24` 访问 `192.168.2.20` 时，以太网帧的目的 MAC 应是谁的 MAC？IP 数据报的目的 IP 是否改变？

### 9. DHCP 与特殊IP地址（选择，2分）

DHCP 客户端尚未获得 IP 地址时，常用的源 IP 和目的 IP 是（ ）。

A. `127.0.0.1` → `8.8.8.8`  
B. `0.0.0.0` → `255.255.255.255`  
C. `192.168.1.1` → `255.255.255.0`  
D. `255.255.255.255` → `0.0.0.0`
