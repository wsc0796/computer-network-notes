# 运输层与网络层：90分高风险变式卷

> 限时：15分钟  
> 题量：6题  
> 过关标准：至少5题正确，且核心步骤不能错误。  
> 核心步骤错误包括：SYN/FIN序号错、cwnd/rwnd混淆、CIDR边界错、最长前缀错、RIP未加1或同一下一跳变差未更新、分片偏移未除以8、MF标志错。

---

## 1. TCP seq/ack 连续变化

客户端 C 初始序号 `1000`，服务器 S 初始序号 `5000`。

1. C 发送 SYN，`seq=1000`。S 回复 `SYN+ACK` 时，`ack` 应是多少？
2. S 的 `SYN+ACK` 使用 `seq=5000`，C 第三次握手 ACK 的 `ack` 应是多少？
3. 握手后 C 发送 300 字节数据，该数据段 `seq` 应是多少？S 确认号是多少？
4. C 紧接着发送不携带数据的 FIN，FIN 的 `seq` 应是多少？S 确认号是多少？

---

## 2. 拥塞控制变式

某 TCP 当前 `cwnd=20 MSS`，`ssthresh=16 MSS`。

1. 当前处于哪个阶段？
2. 此时收到3个重复ACK，按课程简化模型，新的 `ssthresh` 和 `cwnd` 是多少？
3. 后续拥塞避免增长到 `cwnd=12 MSS` 时发生超时，新的 `ssthresh` 和 `cwnd` 是多少？

---

## 3. CIDR边界题

给定地址 `192.168.31.255/20`。

1. 子网掩码是多少？
2. 网络地址和广播地址分别是多少？
3. `192.168.31.255` 能否分配给主机？为什么？

## 4. 最长前缀匹配

路由表如下：

| 目的网络 | 下一跳 |
|---|---|
| 172.16.0.0/12 | A |
| 172.20.0.0/14 | B |
| 172.20.64.0/18 | C |
| 172.20.96.0/20 | D |
| 0.0.0.0/0 | E |

目的地址 `172.20.100.9` 应从哪个下一跳转发？说明匹配依据。

## 5. RIP同一下一跳变差

路由器 R 收到邻居 R2 的 RIP 更新前，路由表如下：

| 目的网络 | 距离 | 下一跳 |
|---|---:|---|
| N5 | 4 | R2 |
| N6 | 5 | R3 |
| N7 | 2 | 直连 |

R2 通告：

| 目的网络 | R2距离 |
|---|---:|
| N5 | 7 |
| N6 | 3 |
| N8 | 1 |

写出更新后的 N5、N6、N7、N8 表项。

## 6. 非整齐数据长度IP分片

一个 IPv4 数据报总长度 `4021` 字节，首部 `20` 字节，`DF=0`，经过 MTU 为 `1000` 字节的链路。

1. 原始数据长度是多少？每片最大数据长度是多少？
2. 分成几片？
3. 写出每片数据长度、MF、片偏移。
