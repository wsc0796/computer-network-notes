# 运输层与网络层：6小时验收卷

> 总用时：20分钟（答题15分钟 + 自评5分钟）
> 规则：闭卷，不允许查看任何资料

## 运输层（第1-6题）

### 1. TCP与UDP对比（填空题，每空1分，共8分）

| 对比项 | UDP | TCP |
|---|---|---|
| 是否面向连接 | (a)________ | 面向连接 |
| 可靠性 | 尽最大努力交付 | (b)________ |
| 数据形式 | (c)________ | 面向字节流 |
| 首部长度 | (d)________ 字节 | 最小 (e)________ 字节 |
| 流量控制 | (f)________ | 有 |
| 拥塞控制 | 无 | (g)________ |
| 通信方式 | 一对一/一对多/多对多 | (h)________ |

### 2. 停止等待与滑动窗口（简答题，10分）

（1）停止等待协议的主要缺点是什么？（3分）

（2）滑动窗口相比停止等待的核心改进是什么？（3分）

（3）写出停止等待、GBN、SR三者的发送窗口大小关系（用 =1, >1 表示）。（4分）

```
停止等待  发送窗口 ___
GBN      发送窗口 ___,  接收窗口 ___
SR       发送窗口 ___,  接收窗口 ___
```

### 3. GBN和SR对比（填空题，每空1分，共6分）

| 对比项 | GBN | SR |
|---|---|---|
| 接收乱序帧 | (a)________ | 缓存 |
| 确认方式 | (b)________ | 独立确认 |
| 重传方式 | (c)________ | 只重传丢失帧 |
| 计时器数量 | (d)________ | 每帧各1个 |
| 最大发送窗口（n位序号） | (e)________ | (f)________ |

### 4. TCP seq/ack计算（计算题，10分）

主机A向主机B发送TCP报文段，seq=200，数据长度300字节。

（1）B正确接收后回复的ack是多少？（3分）

（2）A紧接着发送下一段，seq是多少？（3分）

（3）若该报文段是SYN报文段（SYN=1），seq=200，则B回复的ack是多少？A下一段的seq是多少？（4分）

### 5. TCP拥塞控制（计算题，8分）

某TCP连接当前cwnd=16 MSS，ssthresh=8 MSS。

（1）当前处于什么阶段（慢开始还是拥塞避免）？（2分）

（2）若发生超时重传，新的ssthresh和cwnd分别是多少？（3分）

（3）若收到3个重复ACK（而非超时），新的ssthresh和cwnd分别是多少？之后进入什么阶段？（3分）

### 6. 三次握手/四次挥手（简答题，8分）

（1）为什么TCP连接建立需要三次握手，而不是两次？（4分）

（2）TIME-WAIT的两个作用是什么？等待多长时间？（4分）

## 网络层（第7-12题）

### 7. CIDR计算（计算题，10分）

给定IP地址 192.168.10.130/26：

（1）写出子网掩码的点分十进制形式。（2分）

（2）求该IP所在网络的网络地址。（3分）

（3）求该网络的广播地址。（3分）

（4）该网络有多少个可用主机地址？（2分）

### 8. 等长子网划分（计算题，10分）

某公司使用网段 172.16.0.0/16，需要划分8个等长子网。

（1）需要从主机位借几位？（2分）

（2）新的子网掩码（前缀长度）是多少？（2分）

（3）每个子网有多少个可用主机地址？（3分）

（4）写出第一个子网的网络地址和广播地址。（3分）

### 9. 最长前缀匹配（计算题，10分）

某路由器有以下路由表：

| 目的网络 | 下一跳 |
|---|---|
| 192.168.10.0/24 | R1 |
| 192.168.10.0/27 | R2 |
| 192.168.10.32/28 | R3 |
| 0.0.0.0/0 | R4 |

目的IP为以下地址时，分别从哪个下一跳转发？

（1）192.168.10.20 → ______（3分）

（2）192.168.10.40 → ______（3分）

（3）10.0.0.5 → ______（4分）

### 10. RIP更新（计算题，10分）

路由器R6的路由表如下：

| 目的网络 | 距离 | 下一跳 |
|---|---|---|
| Net1 | 4 | R4 |
| Net2 | 2 | R5 |
| Net3 | 1 | 直接 |

现收到R4发来的路由更新（R4到各网络的距离）：

| 目的网络 | R4的距离 |
|---|---|
| Net1 | 3 |
| Net2 | 4 |
| Net4 | 2 |

请写出R6更新后的路由表（目的网络、距离、下一跳）。

### 11. IP分片（计算题，10分）

一个IP数据报总长度4000字节，首部长度20字节，DF=0。需要经过一条MTU=1500字节的链路。

（1）该数据报需要分成几片？（2分）

（2）写出每片的：数据长度、MF值、片偏移。（8分）

### 12. 综合简答（10分）

（1）跨网络通信时，源主机ARP查询的是谁的MAC地址？为什么？（5分）

（2）NAT的主要作用是什么？（5分）
