# 运输层与网络层：阶段抽查题答案

---

## A. 运输层抽查答案

### 1. TCP/UDP 与分用（2分）

答案：B。

TCP 连接通常由四元组标识：源 IP、源端口、目的 IP、目的端口。UDP 分用也需要端口号，但 UDP 本身无连接。

### 2. 可靠传输分类（2分）

答案：

```text
单帧滑动窗口：停止-等待 ARQ
多帧滑动窗口：GBN、SR
```

补充：数据链路层窗口按帧编号；TCP 窗口按字节编号，二者不能完全等同。

### 3. TCP 序号确认号（3分）

```text
数据段 ack = 2500 + 420 = 2920
FIN 占用 1 个序号，所以 ack = 6000 + 1 = 6001
```

评分：数据 ACK 2分；FIN 占序号 1分。

### 4. RTT/RTO（3分）

```text
RTTVARnew = 0.75*16 + 0.25*|80-120|
          = 12 + 10
          = 22 ms

SRTTnew = 0.875*80 + 0.125*120
        = 70 + 15
        = 85 ms

RTO = 85 + 4*22 = 173 ms
```

评分：每个值 1分。

### 5. 拥塞控制（2分）

答案：错。

正确说法：流量控制由接收方通告窗口 `rwnd` 约束；拥塞控制由拥塞窗口 `cwnd` 约束。实际发送窗口取 `min(rwnd, cwnd)`。

## B. 网络层抽查答案

### 6. 非大题概念组（2分）

答案：错。

数据报网络通信前不建立连接，每个分组独立转发；虚电路网络通信前建立虚电路，后续分组沿同一虚电路转发。

### 7. 转发、路由、直接交付/间接交付（2分）

答案：

```text
转发表/路由表；路由表/转发表；直接；间接
```

评分：每空 0.5分。

### 8. IP 与 MAC（2分）

目的 MAC 是默认网关接口的 MAC。IP 数据报的目的 IP 仍为 `192.168.2.20`，不会因为下一跳转发而改成网关 IP。

### 9. DHCP 与特殊IP地址（2分）

答案：B。

客户端还没有地址时用 `0.0.0.0` 作源 IP，通过受限广播 `255.255.255.255` 寻找 DHCP 服务器。

### 10. CIDR 与最长前缀（2分）

答案：R2。

`172.20.70.5` 同时匹配 `/16` 和 `/18`，最长前缀是 `172.20.64.0/18`，所以选 R2。
