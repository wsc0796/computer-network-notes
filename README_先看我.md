## 文件夹说明

```text
00_使用指南与索引：先看这里，负责告诉你怎么复习。
01_考前冲刺复习：高频考点、TCP/HTTP专题、大题专项、30题速通。
02_刷题模拟卷：模拟卷和详细解析。
03_往年卷线索：江西农业大学计网往年卷线索。
  - 也放了江西财经大学《计算机网络》A卷和对应题型提炼，用来参考传统期末卷题型。
04_课程笔记原文：原始课程笔记，适合补概念。
05_教材PDF：课本PDF，考前只建议查概念。
08_网盘资料接入：使用 Codex Chrome 扩展接入百度网盘资料的分阶段 Prompt、下载目录、提取目录和整理输出目录。
学习通作业：学习通平时作业按五层归档（体系结构/数据链路层/网络层/运输层/应用层），每层含截图+对照分析。
99_图片素材：所有平时作业截图、联系表和原图。
```

## 考前最短路线

```text
先背平时作业高频点。
再背TCP/HTTP专题。
运输层薄弱时看网课截图补强移植。
运输层想达到考研程度时，用运输层考研题型训练记录错题。
再背大题专项。
然后刷模拟卷一。
最后订正错题。
```
