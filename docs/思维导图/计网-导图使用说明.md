# 计网思维导图使用说明

## 先别再背整张大图

你现在的问题不是“知识点不够多”，而是看到题目时调不出答案。新版导图按题型拆：

- 选择/判断：背触发词和陷阱。
- 简答：背标准答题骨架。
- 大题：背步骤模板。
- TCP：背时序图、状态、为什么。

## 2小时用法

| 时间 | 任务 | 文件位置 |
|---|---|---|
| 0:00-0:20 | 选择判断触发词自测 | `计网-高质量思维导图-PlantUML.md` 第2节 |
| 0:20-0:55 | 简答题模板默写 | 第3节 |
| 0:55-1:40 | 网络层计算题模板 | 第4节 |
| 1:40-2:00 | TCP专题默写 | 第5节 |

## 自测规则

1. 看二级节点，不看下面答案。
2. 口头说出标准答案。
3. 说不出来的节点，在纸上抄一遍。
4. 每张图只允许错 3 个；超过 3 个，整张图重背。

## 优先级

1. 先背：网络层计算、TCP、ARP、DNS、五层封装。
2. 再背：CRC、CSMA/CD、交换机、HTTP/HTTPS、DHCP。
3. 最后补：邮件、FTP、物理层细节。
