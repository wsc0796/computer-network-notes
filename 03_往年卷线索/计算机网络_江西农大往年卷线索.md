# 2. 结合你平时作业后的押题方向

你图片里的作业题明显偏基础概念 + 数据链路层计算，因此复习优先级要调整为：

| 优先级 | 内容 | 原因 |
|---|---|---|
| A | OSI/TCP-IP分层、PDU、服务与协议 | 作业反复考 |
| A | 时延计算 | 作业有发送时延、传播时延计算 |
| A | 数据链路层三大问题 | 作业直接考封装成帧、透明传输、差错检测 |
| A | CRC计算 | 作业有生成多项式和余数 |
| A | CSMA/CD与最小帧长 | 作业考争用期、2t、最小有效数据长度 |
| A | 以太网、MAC、交换机/网桥 | 作业集中考MAC地址、MAC表、网桥层次 |
| B | TCP/UDP、HTTP、DNS、POP3 | 往年卷线索常见 |
| B | IP地址、子网掩码、路由器 | 往年卷线索常见 |

# 3. 最短复习路线

```text
第1轮：背 OSI/TCP-IP 分层、PDU、协议/服务区别。
第2轮：练时延计算、传输效率、CRC。
第3轮：背数据链路层：封装成帧、透明传输、差错检测、PPP不可靠。
第4轮：背局域网：CSMA/CD、以太网、MAC地址、交换机/网桥。
第5轮：刷模拟卷，重点看错题解析。
```
