## 一、题型定义

```text
大题：计算题、综合应用题、需要多步推导的题目。
      在考试中通常出现在综合应用题（43-46题）位置。
      特征是：需要公式、计算步骤、画图、多步推理。

非大题：选择题、填空题、判断题、简答题、名词解释。
        特征是：直接判断/记忆/简短回答即可。
```

## 六、易错点标记

以下知识点容易在考试中被混淆，需要特别注意：

```text
运输层易错：
1. TCP面向字节流，UDP面向报文——不要背反！（学习通第7题直接考）
2. 网络层服务类型不影响运输层运行机制，但影响服务质量（学习通第8题）
3. 流量控制看接收方（rwnd），拥塞控制看网络（cwnd）
4. 慢开始是"指数增长"，不是"增长得慢"
5. 超时重传→cwnd降为1；3个重复ACK→快重传+快恢复（cwnd降一半）
6. SYN和FIN各消耗1个序号，纯ACK不消耗序号

网络层易错：
1. ARP不跨网络——跨网时ARP查的是网关MAC，不是目的主机MAC
2. 最长前缀匹配——选前缀最长的，不是第一条匹配的，也不是范围最大的
3. RIP"距离"是跳数，不是带宽或时延
4. IPv6路由器不分片——过大直接丢弃并回ICMPv6 Packet Too Big
5. ICMP封装在IP数据报中，不是运输层协议
6. 网络层是主机到主机，运输层是进程到进程——不要混淆
```
